Detailed Description

Example:

TextArea {
    style: TextAreaStyle {
        textColor: "#333"
        selectionColor: "steelblue"
        selectedTextColor: "#eee"
        backgroundColor: "#eee"
    }
}

Property Documentation

backgroundColor : color

The background color.


read-onlycontrol : TextArea

The TextArea this style is attached to.


font : font

The current font.


renderType : int

Override the default rendering type for the control.

Supported render types are:

  • Text.QtRendering
  • Text.NativeRendering

The default value is platform dependent.

See also Text::renderType.


selectedTextColor : color

The highlighted text color, used in selections.


selectionColor : color

The text highlight color, used behind selections.


textColor : color

The text color.


textMargin : real

The default margin, in pixels, around the text in the TextArea.

This QML property was introduced in QtQuick.Controls.Styles 1.3.

See also TextArea::textMargin.
